STideas

Key ideas

unit

Sentence → one vector

Transformer → pooling → optional L2. Mean (masked) vs CLS vs max — wrong pooling = silent bugs.

metric

util.cos_sim

384-d MiniLM or 768-d mpnet. normalize_embeddings=True ⇒ cosine ≡ IP.

space

Same model both sides

Same checkpoint + same prefixes (E5/BGE query:/passage:). Never mix indexes.

03notes/sentence-transformers.md
Codeencode()

encode() in a few lines

from sentence_transformers import SentenceTransformer, util

model = SentenceTransformer("all-MiniLM-L6-v2")
emb = model.encode(
    ["Hello", "Hi there", "Battery life is awful"],
    normalize_embeddings=True,  # cosine ≡ dot
)
print(util.cos_sim(emb[0], emb[1]))  # ~0.7–0.9 greetings
print(util.cos_sim(emb[0], emb[2]))  # ~0.0–0.3 different topic

Deeperdive

Deeper dive

train

Why SBERT exists

Siamese/triplet fine-tune so cosine matches labels. Vanilla BERT CLS is weak at similarity.

API

encode knobs

batch_size, convert_to_numpy, normalize_embeddings — set True when DB uses IP.

speed

Throughput

MiniLM CPU ~500–2000 short sent/s; GPU often 5–20×. 512-tok chunks ≫ titles.

RAM

Matryoshka

Some models allow first-256 dims. Don’t truncate arbitrary MiniLM unless trained for it.

asym

Q vs passage

QA / MultipleNegativesRankingLoss beats symmetric paraphrase on FAQ search.

eval

Multilingual

Measure Recall@k on your language pair — not only English MTEB headlines.

Case Studylab

Case Study

Build a FAQ retriever for 5 000 support answers (~80–200 tokens) used by a RAG bot.

inputs

Inputs

FAQ strings + ids; SentenceTransformer("all-MiniLM-L6-v2"); queries from a 100-example gold set.

steps

Steps

offline encode(faqs, batch_size=64, normalize_embeddings=True) → 5000×384 matrix (~7.3 MB) → FAISS IndexFlatIP → at query time encode one vector → top-5 → pass texts to the LLM.

output

Output

p50 retrieve <50 ms on CPU for this size; gold Recall@5 target ≥0.85 before wiring RAG.

check

Verify

cosine(Hello,Hi there) ≫ cosine(Hello, unrelated FAQ); no second model mixed into the index; if switching to multilingual FAQs, rebuild the entire index with one multilingual checkpoint.
