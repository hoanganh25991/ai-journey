From text to ID

step 1 · split

"hello friend"

→ ["hello", "friend"] — each piece is a subword token.

step 2 · lookup

token → ID

["hello", "friend"] → [5021, 883]. The model only sees this number sequence.

pitfall

Tokenizer mismatch
