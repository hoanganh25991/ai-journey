From raw scores to probabilities

Softmax stretches the largest logit and forces the vector to sum to 1 — so you can pick argmax and report confidence.

Metricstable

Accuracy is easy — and often misleading

MetricAsksWhen it lies
AccuracyHow often correct?95% majority class → “always majority” looks great
PrecisionOf predicted positives, how many true?Ignores missed rare cases
RecallOf true positives, how many found?Critical for fraud / rare disease
F1Harmonic mean of P & RBetter default on skewed data
10inspect label distribution first
Pitfallswatch

Easy ways to fool yourself

metric

Wrong metric on skew

Report F1 / per-class recall — not only accuracy.

leak

Train / val leakage

Same review in both splits → fake high scores.

noise

Label noise

Inconsistent human labels cap how good any model can get.

threshold

Threshold blindness

Binary default 0.5 may be wrong — tune on validation.
